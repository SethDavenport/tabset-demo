# TabsetDemo

Fractal TabSet Component Demo with Angular, Redux, and @angular-redux/store.

To run:

```sh
git clone https://github.com/SethDavenport/tabset-demo.git
cd tabset-demo
npm i
npm start
```

Open your browser at http://localhost:4200
